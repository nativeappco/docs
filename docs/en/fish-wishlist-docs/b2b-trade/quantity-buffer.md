# Quantity Buffer

> **Trade plan feature.** Available during your 14-day free trial.

Set a minimum stock threshold below which items cannot be added to cart from the wishlist. Protects you from overselling on low-stock products.

* * *

## How to enable

1.  Go to **Fish Wishlist > Settings > B2B Features**
2.  Toggle on **Quantity Buffer** and set your buffer value

* * *

## How it works

The buffer value is the minimum available inventory required before a wishlist item can be added to cart.

**Example:** If your buffer is set to `5` and a product has only 3 units in stock, the Add to Cart button will be disabled for that item on the wishlist.

* * *

## Use cases

*   Prevent customers from adding items you want to hold back for other orders
*   Avoid overselling on products nearing the end of their stock run
*   Works well alongside [Quantity Pickers](/quantity-pickers) — if a customer sets a quantity higher than the buffer allows, the item won't be added
