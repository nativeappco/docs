When a customer qualifies for more than one **automatic store credit** promotion at the same time, Stork Credit uses your **Promotion Stacking** setting to decide how much credit to issue.

## Where to find Promotion Stacking

<m-steps><m-step title="Open Settings">From the <strong>Stork Credit App Dashboard</strong>, open <strong>Settings</strong>.</m-step> <m-step title="Go to Promotion Stacking">On the Settings page, find the <strong>Promotion Stacking</strong> section (next to <strong>Calculation Rules</strong>).</m-step></m-steps>

## Choose how promotions apply

In **Promotion Stacking**, select one of these options:

### Stack promotions

**What it does:** Sums up the credit from _all eligible_ promotions. The customer receives the **total combined value**.

*   Use this when you want multiple promotions to “add up” (for example, a seasonal offer plus a VIP offer).

### Best value only

**What it does:** Compares eligible promotions and applies only the **single promotion with the highest credit value**.

*   Use this when you want to avoid over-rewarding customers when multiple promotions overlap.

<m-warning>Changing <strong>Promotion Stacking</strong> affects how credit is calculated when multiple promotions overlap going forward. Review any overlapping active promotions before switching this setting.</m-warning>

## What counts as “eligible”

A promotion is eligible when the customer’s order matches that promotion’s requirements (for example: products/collections, minimum purchase, customer eligibility, and schedule).

<m-tip>If you run many overlapping promotions, <strong>Best value only</strong> is the simplest way to prevent unexpected combined payouts while still ensuring customers get the strongest offer available.</m-tip>
